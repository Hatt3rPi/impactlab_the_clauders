# The Clauders — Wiki del equipo

> *"5 millones de chilenos tienen derechos financieros que no logran entender."*
> — Claude Impact Lab Chile 2026

Bienvenidos al espacio de trabajo del equipo **The Clauders** para el [Claude Impact Lab Chile 2026 — *El puente al ciudadano*](https://fintech.benditaia.cl/es), 48 horas de hackathon organizado por **Anthropic, BenditaIA y FinteChile** los días **6 y 7 de mayo de 2026** en Espacio Riesco, dentro del Chile Fintech Forum.

---

## Resumen ejecutivo

| | |
|---|---|
| **Competencia** | Claude Impact Lab Chile 2026 — El puente al ciudadano |
| **Organizador** | Anthropic + BenditaIA + FinteChile |
| **Fechas del lab** | 6-7 de mayo de 2026 (48 h) |
| **Lugar** | Espacio Riesco, Santiago — Chile Fintech Forum 2026 |
| **Líneas temáticas** | Inclusión financiera · Ciberseguridad ciudadana · Protección de datos |
| **Modalidad** | Equipos de 2-4 personas (también solo) |
| **Stack obligatorio** | Claude API, Claude Code, Agent SDK, MCPs |
| **Datasets** | CMF, SII, SERNAC (datos públicos) |
| **Pitch final** | 7 de mayo a las 12:00 |
